Swiggy Sales Analysis Dashboard

This project analyzes Swiggy sales data using an interactive Excel dashboard. The dashboard helps identify sales trends, customer ordering behavior, food category performance, and city-wise growth through data visualization and analytical reporting.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning
Key Insights
Total Sales: ₹53.01M
Average Rating: 4.34
Total Orders: 197.43K
Non-Veg category generated the highest sales
Weekend sales were higher than weekdays
Bengaluru was the top-performing city
Skills Used

Data Cleaning, Data Visualization, Dashboard Designing, Business Analysis, Excel Analytics
